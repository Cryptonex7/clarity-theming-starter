SASS-based theming is deprecated in Clarity 3.0. This means that it will no longer be enabled in Clarity version 4 and later. See the Clarity theming documentation for guidance on how to migrate from SASS-based theming in Clarity to CSS custom property theming.
SASS Based Theming

Overriding SASS variables in Clarity requires that you compile the Clarity SASS within your own project and disable CSS custom properties in the Clarity SASS-to-CSS output.
